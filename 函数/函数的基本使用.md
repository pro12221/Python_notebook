## 函数的定义
函数的定义只检测语法不执行代码
```python
def 函数名(参数1,参数2,...):
    """
    函数，参数，返回值的简单介绍
    """
    代码块
    return
```

## 定义函数的过程
```
x=10 # 定义变量
print(x) # print打印的不是变量的内存地址而是值

def login():
    x=10
    y=20
print(login) # 打印的是函数的内存地址
```

## 定义函数的三种形式
```python
# 无参函数
def say_hello():
    print("="*50)
    print("hello")
say_hello()

# 有参函数
def add_num(x,y):
    print(x+y)
add_num(1,2)

# 空函数
def null_func():
    pass
```

## 函数的参数使用
```python
# 位置参数,定义函数的时候按照从左到右顺序依次定义
def foo(x,y,z):
    print(x,y,z)
foo(1,2,3)

# 关键字参数，在调用函数时按照key=value的形式为参数传值
foo(x=111,z=222,y=123)

# 默认参数,可穿可不穿
def register(name,age,gender="male"):
    print(name,age,gender)
register("xx",12,"female")
```
