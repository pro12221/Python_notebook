## 迭代器
### 可迭代对象
```python
#但凡内置有_iter_方法的都是可迭代对象,调用可迭代对象可以将可迭代对象转换为迭代器对象
# 有next方法的才是迭代器对象，可迭代对象有iter方法就行
dic={"a":1,"b":2,"c":3}
dict_iterable=dic.__iter__() # 调用iter方法
print(dict_iterable)
print(type(dict_iterable))
print(dict_iterable.__next__()) # 获取字典的key

# 函数实现例子
def foo(iterable):
    x=iterable.__iter__()
    while True:
        try:
            print(x.__next__())
        except StopIteration:
            break

dict1={'a':1,'b':2,'c':3,'d':4}
list=[1,2,3,4,5]
foo(dict1)
foo(list)
```
