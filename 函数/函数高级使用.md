## 函数对象
### 取代多分支例子
```python
# 小例子
def login():
    print('from login...')


def withdrawal():
    print("from withdrawal....")


def transfer():
    print("from transfer....")


def recharge():
    print("from recharge...")

def xxx():
    print("xxx")

dic = {
    "1": [login,"登录"],
    "2": [withdrawal,"提现"],
    "3": [transfer,'转账'],
    "4": [recharge,"充值"],
    "5": [xxx,"xxx功能"],
}

# dic["1"]()
# dic["2"]()
# dic["3"]()

while True:
    for k in dic:
        print("%s %s" %(k,dic[k][1]))
    choice = input("请输入命令编号：").strip()
    if choice not in dic:
        print("必须输入正确的命令编号！")
        continue
    dic[choice][0]()
```

###  函数的嵌套
#### 函数的嵌套调用
```python
def max2(x,y):
    if x>y:
        return x
    else:
        return y
def max4(a,b,c,d)
    res1=max2(a,b)
    res2=max2(res1,c)
    res3=max2(res2,d)
```
