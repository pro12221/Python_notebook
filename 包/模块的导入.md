| 语法                             | 解释                        | 示例                                    |
| ------------------------------ | ------------------------- | ------------------------------------- |
| `import module`                | 导入整个模块，需要通过模块名访问模块内容      | `import math`<br>`math.sqrt(16)`      |
| `from module import something` | 从模块中导入特定的内容，直接使用该内容       | `from math import sqrt`<br>`sqrt(16)` |
| `from module import *`         | 导入模块中的所有内容，直接访问模块中的所有公共部分 | `from math import *`<br>`sqrt(16)`    |


## 绝对路径和相对路径导入
```python
# 模块搜索路径 是由 sys.path 确定的，通常包括以下几个位置：
# 当前目录
# 标准库的目录
# 安装的第三方库目录（例如通过 pip 安装的库）


# 假设包结构是：
# mypackage/
# ├── __init__.py
# ├── module1.py
# └── subpackage/
#     └── module2.py

# 绝对导入
from mypackage.module1 import function1
from mypackage.subpackage.module2 import function2

# 相对导入
from .module1 import function1  # 导入同一包中的 module1
from ..subpackage.module2 import function2  # 导入上级包中的 module2

```
