## call方法
```python
# __call__是一个特殊方法（魔术方法），它允许类的实例像函数一样被调用。
class CallableClass:
    def __init__(self, name):
        self.name = name

    def __call__(self, *args, **kwargs):
        print(f"实例 {self.name} 被调用!")
        print(f"参数: args={args}, kwargs={kwargs}")
        return f"Hello, {self.name}!"


# 创建实例
obj = CallableClass("示例对象")

# 像函数一样调用实例
result = obj("参数1", "参数2", key="value")
print(result)
```
## exec函数
exec()是一个内置函数，用于动态执行Python代码（字符串形式）。
```python
# exec(source, globals=None, locals=None)
# 1. 执行简单代码
code = """
x = 10
y = 20
result = x + y
print(f"结果是: {result}")
"""
exec(code)

# 2. 使用全局和局部命名空间
global_namespace = {'a': 100}
local_namespace = {'b': 200}

code = """
c = a + b + 50
print(f"c = {c}")
"""

exec(code, global_namespace, local_namespace)
print("局部命名空间:", local_namespace)
```
